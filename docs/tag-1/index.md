!!! note "Tag 1 ist Frontend-only"
    An Tag 1 geht es **bewusst nur** um HTML und CSS. Kein JavaScript, kein `fetch()`, kein Backend. Das kommt ab Tag 2. Die Übung am Vormittag und das Projekt am Nachmittag bauen eine **rein visuelle** Webseite auf.
